Разработать приложение на С++ с использованием библиотеки Qt для подсчета контрольных сумм файлов.

Требования:

1. Приложение выполняется в консольном режиме.
2. Приложение сканирует заданный стартовый каталог и вложенные в него каталоги на наличие файлов и рассчитывает контрольную сумму каждого файла. Стартовый каталог задается через параметр командной строки. Контрольная сумма рассчитывается по алгоритму MD5.
3. Приложение выводит в консоль полный путь к файлу и его контрольную сумму. Путь и имена файлов при выводе должны быть упорядочены в алфавитном порядке.
4. Для расчета контрольной суммы каждого файла используется отдельный поток.
5. Реализовать два варианта приложения с синхронизаций данных через сигналы/слоты и через мьютексы.
6. Приложение должно корректно обрабатывать отсутствие прав доступа к файлам и каталогам.
7. Приложение должно собираться и выполняться в операционных системах Windows и Linux.
8. Наличие warning-ов при сборке недопустимо.
